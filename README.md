# 📂 File Organizer UI

A modern and interactive **File Organizer Web App** built using **HTML, CSS, and JavaScript**.
This project simulates a basic file management system like a desktop file explorer.

---

## 🚀 Live Demo

🔗https://file-organizer-web.netlify.app/

---

## 📂 GitHub Repository

🔗https://github.com/aayushgupta-4341/File-Organizer.git

---

## ✨ Features

* 📁 Create folders
* 📄 Create files
* ✏️ Rename files & folders (editable text)
* 🗑️ Delete files & folders
* 🖱️ Drag & Drop files into folders
* 📂 Double-click to open folders
* ⬅️ Navigate back to root
* 🎯 Dynamic folder structure (real-time update)
* 📱 Responsive UI
* 🎨 Clean UI with background support
